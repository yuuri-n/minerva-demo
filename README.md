# Minerva CRM デモ

## 起動方法

`demo/index.html` をブラウザで開くだけで動作します（外部依存なし）。

## 画面構成

| ファイル | 画面名 |
|---|---|
| demo/index.html | ホームダッシュボード |
| demo/leads.html | 架電リスト |
| demo/opportunity.html | 商談詳細（活動記録・タイムライン） |
| demo/case.html | ケース詳細（関連商談） |
| demo/screen_list.html | 画面一覧・遷移図・更新ルール |

## テストシナリオ

### シナリオ1：いま忙しい
1. `leads.html` を開く
2. 任意の商談の「📞 架電記録」ボタンをクリック
3. コール結果「いま忙しい」、次回架電日時を入力して保存
4. → 最新ステータスが「折り返し待ち」、次アクションが「再架電」に更新
5. → タイムラインに記録が追加される

### シナリオ2：アポ獲得
1. コール結果「アポ獲得」、MTG日時を入力して保存
2. → 最新ステータスが「アポ確定」に更新
3. → ステージが1段階前進（アプローチ中→商談中）

## データリセット
右下の「🔄 データリセット」ボタンでlocalStorageをクリアし、初期データに戻ります。
